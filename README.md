# Movie-Search-App
A movie search interface powered by the OMDb API.

🚀 Features

• Search movies by title
• Display movie results
• API integration


🧩 Tech Stack

• HTML
• CSS
• JavaScript
• OMDb API


📂 Project Structure

index.html
style.css
script.js


📸 Preview

Shows a search bar and a list of movie results.

🧠 How It Works

The app sends a request to OMDb using the user’s search query and displays the returned movie list.
